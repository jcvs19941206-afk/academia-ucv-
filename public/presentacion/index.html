El Sistema Definitivo de Control Académico

Resumen de Desarrollo e Implementación

El Reto Inicial 🚀

Crear un Dashboard académico, moderno y escalable.

  • Panel de control interactivo (Cursos y Tareas).
  • Autenticación segura.
  • Interfaz atractiva y responsiva (Mobile & Desktop).
  • Despliegue rápido en la nube.

Stack Tecnológico 🛠️

Para garantizar rendimiento y modernidad, elegimos:

Frontend

  • Next.js 15 (App Router) para React súper rápido.
  • Tailwind CSS para estilos ágiles.
  • Shadcn UI para componentes premium.

Backend & Hosting

  • Supabase para base de datos y autenticación.
  • Vercel para despliegue automático.

Paso 1: Diseño del Frontend 🎨

Se crearon las vistas principales priorizando la estética.

  • Pantalla de Login / Registro con video de Bienvenida UCV integrado.
  • Dashboard Principal: Resumen de actividades.
  • Módulo de Cursos: Permite gestionar, duplicar o editar cursos.
  • Módulo de Tareas: Todo sincronizado en una UI fluida.

Paso 2: Conexión con Supabase 🔐

Integramos Supabase SSR (Server-Side Rendering).

  • Configuramos NEXT_PUBLIC_SUPABASE_URL y la ANON_KEY.
  • Se conectó el formulario de Login y Registro directamente a la DB.
  • Se solucionó el problema de "Error al iniciar sesión" agregando un fallback seguro de entorno.

Paso 3: Batalla contra el Error 500 ⚔️

El desafío más grande: El servidor fallaba en Producción.

  • Vercel inyectaba middlewares no deseados de configuraciones antiguas (Sentry).
  • Un comentario de JSX ({/* */}) rompió el compilador de Next.js ocultando los errores.
  • Solución: Limpiamos configuraciones, eliminamos vercel.json y depuramos la sintaxis. ¡Vercel pasó a verde! ✅

El Resultado Final 🌟

Una aplicación en tiempo real, 100% desplegada y lista para usarse.

Visitar AcademIA Oficial (https://academia-ucv.vercel.app)

* Recuerda mantener siempre al día tus variables de entorno en Vercel.
